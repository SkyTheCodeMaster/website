devbin

Provides a devbin api to interact with devbin without using the devbin shell command.

get(id)Get a string from devbin.
put(string [, name])Put a string on devbin.
get(id)Source

Get a string from devbin.

Parameters

  1. id string The paste id that you want to download.

Returns

  1. string | nil The string containing the paste, or nil.
  2. nil | string The reason why it couldn't be retrieved.
put(string [, name])Source

Put a string on devbin.

Parameters

  1. string string The string that you want to put on devbin.
  2. name? string The name of the paste, defaults to "CC:T Paste".

Returns

  1. string | nil A string containing the id of the paste.
