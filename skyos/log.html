log

The log library is a library to store logs for various programs. It will keep the logs down to a certain size.

create(file)Create returns a table of functions for writing to a log file.
create(file)Source

Create returns a table of functions for writing to a log file.

Parameters

  1. file string Path to the log file.

Returns

  1. table log Logwith functions to write to the log file.

Types

log

Log object, has functions for writing to the log.

log:save()Source

Save the log, writing all the written info to the file.

log:close()Source

Close the log, stopping further use.
